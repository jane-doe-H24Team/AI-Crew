How to Set Up a Telegram Bot for AI-Crew

This guide will walk you through creating a Telegram bot and getting the necessary API token to use with the AI-Crew application. The process is managed entirely through a special bot called the BotFather.

Step 1: Talk to the BotFather

The BotFather is Telegram's official bot for creating and managing all other bots.

  • Open your Telegram app (desktop or mobile).
  • In the search bar, search for the user @BotFather. Make sure to select the official one with a blue checkmark next to its name.
  • Start a chat with the BotFather and send the command: /start.

Step 2: Create Your New Bot

Now, you will ask the BotFather to create a new bot for you.

  • In the chat with the BotFather, send the command: /newbot.
  • The BotFather will then ask you for two things in sequence:
    1. A display name: This can be anything you like (e.g., "AI-Crew Test Avatar").
    2. A username: This must be unique and, most importantly, it must end in "bot". For example: AICrewTestBot or ai_crew_test_bot. If the username is already taken, you will have to try a different one.

Step 3: Get the Bot Token

This is the most important step, where you receive your credential.

  • Once you provide a valid username, the BotFather will congratulate you and send a message containing your bot's API Token.
  • The token is a long string of numbers and letters, like 1234567890:ABCdEfgHiJkLmNoPqRsTuVwXyZ-12345.
  • This token is a secret. Copy it carefully. This is the value you need for the AVATAR_NAME_TELEGRAM_TOKEN variable in your .env file.

Step 4: Start a Conversation with Your Bot

Unlike Discord, you don't need to invite your bot anywhere. You just need to find it and send it a message to initialize the chat.

  • In your Telegram app's search bar, search for the username of the bot you just created (e.g., @AICrewTestBot).
  • Click on it in the search results to open a chat window.
  • Click "Start" or send any message (e.g., "Hello") to begin the conversation.

Step 5: Run the Test

You are now ready to test the integration.

  • Ensure the token you received is correctly saved in your .env file.
  • Start (or restart) your AI-Crew application.
  • Send a new message to your bot in the Telegram chat.
  • When the next scheduled_telegram_check runs (or you trigger it manually), the avatar should receive your message and reply in the chat.
