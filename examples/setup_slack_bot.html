How to Set Up a Slack Bot for AI-Crew

This guide will walk you through creating a Slack App and getting the necessary Bot Token to use with AI-Crew.

Step 1: Create a Slack App

  • Go to the Slack API page and click "Create New App".
  • Choose "From scratch".
  • Give your app a name (e.g., "AI-Crew Avatar") and select the Slack workspace you want to add it to.
  • Click "Create App".

Step 2: Add Permissions (Scopes)

You need to give your bot the correct permissions to read and write messages.

  • From your app's settings page, go to "OAuth & Permissions" in the left sidebar.
  • Scroll down to the "Scopes" section. Under "Bot Token Scopes", click "Add an OAuth Scope".
  • Add the following scopes one by one:
    • channels:history (to read messages in public channels)
    • groups:history (to read messages in private channels)
    • chat:write (to send messages)
    • channels:read (to get a list of public channels)
    • groups:read (to get a list of private channels)
    • auth.test (to get the bot's own user ID)

Step 3: Install the App and Get the Token

  • At the top of the "OAuth & Permissions" page, click "Install to Workspace".
  • Follow the prompts to authorize the app.
  • After installation, you will be redirected back to the "OAuth & Permissions" page. You will now see a "Bot User OAuth Token" that starts with xoxb-.
  • Copy this token. This is the value for your AVATAR_NAME_SLACK_BOT_TOKEN variable in your .env file.

Step 4: Invite the Bot to a Channel

  • Open your Slack workspace.
  • Go to a channel where you want the bot to be active.
  • In the message box, type /invite @your-bot-name (e.g., /invite @AI-Crew Avatar) and press Enter.

Step 5: Run the Test

  • Ensure your AI-Crew application is running with the correct Slack token in the .env file.
  • Send a message in the channel you invited the bot to.
  • When the next scheduled_slack_check runs (or you trigger it manually), the avatar should post a reply in the channel.
