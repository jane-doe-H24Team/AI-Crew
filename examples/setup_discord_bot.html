How to Set Up a Discord Bot for AI-Crew

This guide will walk you through creating a Discord Bot and getting the necessary credentials to use with the AI-Crew application.

Step 1: Create the Discord Application and Bot

First, you need to register a new application with Discord to get your bot's credentials.

  • Go to the Discord Developer Portal and log in.
  • Click the "New Application" button in the top-right corner and give it a name (e.g., "AI-Crew Avatar").
  • Navigate to the "Bot" tab in the left-hand menu.

  • Crucial Step: Under the "Privileged Gateway Intents" section, you must enable two toggles:

    • SERVER MEMBERS INTENT
    • MESSAGE CONTENT INTENT

    The bot will not work without these permissions.

  • Click the "Reset Token" button (or "View Token") to reveal your bot's token. This is a secret credential.
  • Copy the token. This is the value you need for the AVATAR_NAME_DISCORD_TOKEN variable in your .env file.

Step 2: Create a Private Server for Testing

You need a private space where you and your bot can interact.

  • In your Discord app, click the `+` icon in the server list on the left.
  • Choose "Create My Own", then "For me and my friends".
  • Give your server a name (e.g., "AI-Crew Testing") and click "Create".

Step 3: Invite the Bot to Your Server

Now you need to generate an invitation link to add the bot to the server you just created.

  • Go back to the Discord Developer Portal page for your application.
  • Navigate to the "OAuth2" tab and then click on "URL Generator".
  • In the "Scopes" section, check the box for bot.
  • A new "Bot Permissions" section will appear below. Check the following boxes:
    • Send Messages
    • Read Message History
  • Scroll to the bottom and copy the **Generated URL**.
  • Paste this URL into your web browser and press Enter.
  • An authorization screen will appear. Select the test server you just created from the dropdown menu and click "Continue" and then "Authorize".

Step 4: Run the Test

You are all set! After completing the steps above, you can start testing.

  • Ensure your AI-Crew application is running with the correct Discord token in the .env file.
  • Open your test server in Discord. You should see your bot in the user list on the right.
  • Go to any text channel (e.g., #general) and send a message.
  • When the next scheduled_discord_check runs (or you trigger it manually), the avatar should post a reply in the channel.
