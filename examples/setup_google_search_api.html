How to Set Up the Google Search API for AI-Crew

To enable a reliable web search tool, we will use Google's official Custom Search API. This requires two credentials: an API Key and a Search Engine ID.

Part 1: Get the Search Engine ID

  1. Go to the Programmable Search Engine control panel and sign in with your Google account.
  2. Click the "Add" button to create a new search engine.
  3. In the "What to search?" section, select the option "Search the entire web".
  4. Give your search engine a name (e.g., "AI-Crew Search") and click "Create".
  5. After creation, you will be on the main control panel. In the "Basic Information" section, find and copy the Search engine ID. It will look something like `a1b2c3d4e5f6g7h8i`.

Part 2: Get the API Key

  1. Go to the Google Cloud Console - Custom Search API page.
  2. If prompted, select or create a new project.
  3. Click the "Enable" button to enable the Custom Search API for your project.
  4. Once enabled, go to the Credentials page.
  5. Click "+ CREATE CREDENTIALS" at the top and select "API key".
  6. A new API key will be created. Copy this key. It's a long string of characters.
  7. Important: For security, it's highly recommended to restrict your API key. Click "Edit API key" and under "API restrictions", select "Restrict key" and choose the "Custom Search API".

Part 3: Configure Your .env File

You now have two credentials. Add them to your .env file. These keys are global for the application, not specific to one avatar.

# ... your other variables ...

# --- Google Custom Search API ---
GOOGLE_SEARCH_API_KEY="your_api_key_here"
GOOGLE_SEARCH_ENGINE_ID="your_search_engine_id_here"

Once you have configured these variables, the `web_search` tool will be fully operational and reliable.
